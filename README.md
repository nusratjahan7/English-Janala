# 📘 English Janala

## 📌 Project Description
English Janala is an interactive English learning web application designed to help users improve their vocabulary, grammar, and communication skills in a simple and engaging way.  

This project focuses on making English learning easier for beginners by providing structured lessons, categorized words, and real-life usage examples.

## 🚀 Technologies Used
- HTML  
- CSS  
- JavaScript  
- API Integration  
- Git & GitHub  

## ✨ Key Features

### 📖 Vocabulary by Levels
Users can explore words based on different levels, making learning step-by-step and beginner-friendly.

### 🔍 Word Details & Meaning
Each word includes detailed information, helping users understand meaning, usage, and context properly.

### ⚡ Dynamic Data Loading
Words and levels are fetched dynamically from API, ensuring real-time data handling and smooth user experience.
